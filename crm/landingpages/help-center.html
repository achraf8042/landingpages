Getting Started

Installation

  1. Download the DigiCare EMR installer from your purchase confirmation email
  2. Run the installer and follow the on-screen instructions
  3. Launch DigiCare EMR from your desktop or start menu
  4. Complete the initial setup wizard with your clinic information
  5. Create your administrator account
Note: DigiCare EMR is an on-premise application. All data is stored locally on your computer.

First Time Setup

When you launch DigiCare EMR for the first time, you'll be guided through:

  • Clinic Information: Enter your clinic name, address, and contact details
  • Provider Profile: Add your name, specialty, and license information
  • System Preferences: Configure date format, time zone, and privacy settings

Patient Management

Adding a New Patient

  1. Navigate to the Patients section from the sidebar
  2. Click + Add Patient button
  3. Fill in patient information:
    • First Name and Last Name (required)
    • Date of Birth
    • Gender
    • Contact information (phone, email)
    • Address
  4. Click Save to add the patient

Managing Allergies

Track patient allergies to ensure safe prescribing:

  1. Open patient details by clicking View in the patient table
  2. Navigate to the Allergies tab
  3. Click + Add Allergy
  4. Enter allergen, reaction type, severity, and notes
  5. Save the allergy record

Recording Vital Signs

Track patient vital signs for each visit:

  • Blood Pressure (systolic/diastolic)
  • Heart Rate (bpm)
  • Temperature (°F or °C)
  • Weight (lbs or kg)
  • Oxygen Saturation (%)

Appointment Scheduling

Creating an Appointment

  1. Go to Appointments section
  2. Click + New Appointment
  3. Select patient from the dropdown
  4. Choose date and time
  5. Enter appointment reason
  6. Save the appointment

Appointment Status

Track appointments with status indicators:

  • SCHEDULED: Appointment confirmed
  • COMPLETED: Patient visited
  • CANCELLED: Appointment cancelled
  • NO-SHOW: Patient did not attend

Dashboard View

The dashboard shows:

  • Today's appointments count
  • This week's appointments count
  • Upcoming appointments list with filters (Today, This Week, This Month)

Clinical Notes

Creating Clinical Notes

  1. Navigate to Clinical Notes
  2. Click + New Note
  3. Select patient and appointment (optional)
  4. Document clinical information:
    • Chief Complaint
    • History of Present Illness
    • Physical Examination
    • Assessment/Diagnosis
    • Treatment Plan
  5. Save the note

Viewing Clinical Notes

Click View in the notes table to see a formatted clinical document with:

  • Patient demographics
  • Provider information
  • Complete clinical documentation

Exporting to PDF

Generate professional PDF documents:

  1. Open the clinical note document view
  2. Click Export to PDF
  3. Choose save location
  4. PDF is generated with formatted layout

E-Prescriptions

Creating a Prescription

  1. Go to Prescriptions section
  2. Click + New Prescription
  3. Select patient
  4. Add medications:
    • Medication name
    • Dosage (e.g., 500mg)
    • Frequency (e.g., twice daily)
    • Duration (e.g., 7 days)
    • Special instructions
  5. Add multiple medications as needed
  6. Add general instructions (optional)
  7. Save the prescription

Allergy Warnings

The system will alert you if the patient has recorded allergies. Always review allergy information before prescribing.

Generating Prescription PDF

  1. Open the prescription from the list
  2. Click Export to PDF
  3. Professional prescription document is generated with:
    • Patient details
    • Provider information and signature
    • Medication list with complete details
    • Date and prescription ID

System Requirements

Minimum Requirements

  • Operating System: Windows 10 or later, macOS 10.15+, or Linux (Ubuntu 20.04+)
  • Processor: Intel Core i3 or equivalent
  • RAM: 4GB minimum (8GB recommended)
  • Storage: 500MB for application, additional space for data
  • Display: 1280x720 minimum resolution

Data Storage

DigiCare EMR stores all data locally in:

  • Windows: C:\Users\[Username]\AppData\Local\DigiCareEMR\
  • macOS: ~/Library/Application Support/DigiCareEMR/
  • Linux: ~/.local/share/DigiCareEMR/
Security: Ensure you have proper backup procedures in place for your data directory.

Frequently Asked Questions

Q: Is internet connection required?
No. DigiCare EMR is an on-premise application that works completely offline. All data is stored locally on your computer.
Q: How do I backup my data?
Simply copy the data directory to an external drive or cloud storage. We recommend daily backups. The data directory location is listed in System Requirements above.
Q: Can I use this on multiple computers?
Your license covers one installation. The database is local to each installation. If you need to sync data across devices, contact us about multi-device options.
Q: What if I forget my password?
Use the "Forgot Password" option on the login screen. You'll need to verify your clinic name for password reset.
Q: Are software updates free?
Yes! Your lifetime license includes all future software updates and improvements at no additional cost.
Q: How do I export my data?
Clinical notes and prescriptions can be exported to PDF. For complete data export, the database file can be accessed directly in the data directory.
